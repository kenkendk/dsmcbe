This application searches for the k nearest neighbours (kNN).

The input variables are
k: the number of nearest neighbours, default 10
D: the dimensions to simulate, default 72
N: the number of elements in each dimension, default 1000

You can run the program on a single machine with one of these commands:
./PPU <number of spu's> 
./PPU <number of spu's> <k> <D> <N>
